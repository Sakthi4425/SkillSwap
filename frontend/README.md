# SkillSwap Frontend

Modern React frontend for the SkillSwap skill-sharing platform.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm start
```

The app will open at http://localhost:3000

### Build for Production

```bash
npm run build
```
